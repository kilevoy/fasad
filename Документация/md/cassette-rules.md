# Правила по фасадным кассетам

Этот файл хранит рабочие правила для калькулятора фасадных кассет.
Сюда заносится подтвержденная информация по типам кассет, размерам,
стандартности, доборным позициям, швам, крепежу и связи с подсистемой.

## Общие принципы

- `L` и `H` — габаритные размеры кассеты.
- `Lp` и `Hp` — монтажные размеры.
- Проверка допустимости размера по СТО и проверка стандартности — это разные проверки.
- Позиция из прайса со словом `доборная` — доборная кассета.
- Позиция из прайса со словом `угловая` — угловая кассета.
- Обычная позиция без слов `доборная` и `угловая` — рядовая кассета.

## Общий принцип стандартной кассеты

- Для любого типа кассет стандартной считается кассета, у которой хотя бы одна сторона `L` или `H` совпадает со значением из строки таблицы СТО с примечанием `стандарт`.
- Смысл стандартной кассеты: она изготавливается из рулонной заготовки без отхода.
- Если ни одна сторона не совпадает со значением из строки `стандарт`, кассета не считается стандартной.

## Классификация по типу крепления и характеру руста

### Что уже подтверждено

- `КФ-1` относится к кассетам с видимым креплением.
- `КФ-2` относится к кассетам со скрытым креплением.
- `КФ-3` относится к кассетам со скрытым креплением.
- Для `КФ-1` характерны более широкие и акцентированные швы.
- Для `КФ-2` характерны более узкие швы по сравнению с `КФ-1`.

### Рабочая классификация для калькулятора

- `КФ-1` — `visible fastening`
- `КФ-2` — `hidden fastening`
- `КФ-3` — `hidden fastening`
- `КФ-4` — рабочая классификация как `visible fastening`, но монтажные правила еще нужно добрать до полного подтверждения

### По эстетике фасада

- `КФ-1` и `КФ-4` — кассеты с более выраженным, широким рустом
- `КФ-2` и `КФ-3` — кассеты с более узким и визуально спокойным рустом

### Что учитывать в интерфейсе

Для выбора кассеты полезно хранить:

- `fasteningType = visible | hidden`
- `rustStyle = wide | narrow`
- `mountingComplexity = basic | advanced`

### Что еще нужно подтверждать отдельно

- полный монтажный набор правил для `КФ-4`
- окончательные численные правила по швам `КФ-3` из прямого первоисточника, если мы хотим использовать их как окончательные инженерные нормы

## КФ-1

### Назначение

- Кассета фасадная типа 1.
- Крепеж видимый.
- Боковые края соседних кассет накладываются друг на друга.
- Отверстия соседних кассет совмещаются, и крепление выполняется в общий стык.

### Размеры

- `L/H` — габаритные размеры кассеты.
- `L1/H1` — размеры лицевой плоскости внутри отгибов.
- `Lp` — расстояние между центрами отверстий; для `КФ-1` это шаг вертикальных направляющих.

### Допустимые размеры по СТО

Граничные комбинации:

1. `297 x 297`
2. `297 x 1897`
3. `1897 x 297`
4. `1497 x 1197`

Вывод:

- Для `КФ-1` область допустимых размеров не прямоугольная.
- Проверка должна идти по допустимым сочетаниям, а не по простому диапазону.

### Стандартная кассета

Стандартная строка таблицы:

- `572 x 572`
- крупный стандартный размер: `1197 x 1197`

Правило:

- `КФ-1` стандартная, если `L = 572` или `H = 572`.
- `КФ-1` стандартная, если `L = 1197` или `H = 1197`.

Примеры:

- `572 x 572` — стандартная
- `1197 x 1197` — стандартная
- `572 x 1197` — стандартная
- если ни `L`, ни `H` не равны `572` или `1197`, кассета не стандартная

### Почему размер стандартный

- Для кассеты `572 мм` требуется заготовка `625 мм`.
- `625 + 625 = 1250 мм`.
- Рулон шириной `1250 мм` раскладывается без остатка.
- Для размера `1197 мм` заготовка с учетом отгибов также дает полный стандартный раскрой на `1250 мм`.

### Доборная и угловая

- Если в прайсе есть слово `доборная`, это доборная кассета.
- Если размеры не попадают в стандартные производственные размеры `572/1197`, кассета относится к нестандартной доборной группе.
- Если в прайсе есть `угловая` или `УКФ1`, это угловая кассета.
- Угловая кассета не является стандартной рядовой.

### Швы

Подтверждено инструкцией:

- вертикальный шов регулируемый: `28-32 мм`
- горизонтальный шов: `25 мм`

### Крепеж

- Нельзя считать крепеж как `4 самореза на кассету x количество кассет`.
- Отверстия соседних кассет совмещаются.
- Один саморез работает в общем стыке соседних кассет.
- Для `КФ-1` крепеж нужно считать по швам, а не по изолированной кассете.
- Рабочая формула калькулятора:
- `вертикальные линии крепления = количество кассет по ширине + 1`;
- `крепеж по одному ряду = количество отверстий КФ-1 по высоте`;
- для кассет менее `800 мм` принимать `4` отверстия;
- итог: `вертикальные линии крепления * крепеж по рядам * количество фасадов`.

### Монтажные правила

- Центр овального отверстия для крепления должен располагаться по середине вертикального профиля несущей подсистемы.
- `Lp` является шагом вертикальных направляющих.
- Овальные отверстия дают допуск по ширине вертикального шва.

## КФ-2

### Назначение

- Кассета фасадная типа 2.
- Вертикальный шов формируется через направляющую подсистемы.
- Для нижнего ряда используется начальная планка `ПН`.

### Размеры

- `L` — габаритная ширина.
- `H` — габаритная высота.
- `Lp` — монтажный размер по ширине.
- `Hp` — монтажный размер по высоте.

### Допустимые размеры по СТО

Граничные комбинации:

1. `360 x 200`
2. `360 x 1845`
3. `1860 x 200`
4. `1180 x 1175`

### Стандартная кассета

Стандартные строки:

- малый стандарт: `555 x 550`
- большой стандарт: `1180 x 1175`

Правило:

- `КФ-2` стандартная, если `L = 555` или `H = 550`.
- `КФ-2` стандартная, если `L = 1180` или `H = 1175`.

Примеры:

- `555 x 550` — стандартная
- `1180 x 1175` — стандартная
- `555 x 1175` — стандартная
- `1180 x 550` — стандартная
- если ни `L`, ни `H` не совпадают со стандартными сторонами, кассета не стандартная

### Доборная и угловая

- Если в прайсе есть `доборная`, это доборная кассета.
- Если позиция рядовая, но размеры не попадают в стандартные, это нестандартная доборная логика.
- Если в прайсе есть `угловая`, это угловая кассета.

### Швы

Подтверждено инструкцией:

- вертикальный шов до `20 мм` при монтаже на профиль `ПП 60x27`
- вертикальный шов до `35 мм` при монтаже на `НПШ20`
- горизонтальный шов `20 мм`

### Крепеж и отверстия

- Базово `4 отверстия`.
- При `L > 799 мм` добавляется центральное отверстие.
- Тогда получается `5 отверстий`.

### Подсистема

- Дном вертикального шва является направляющая.
- Направляющая в зоне шва должна иметь тот же цвет, что и кассета.
- Это решается либо окрашенной полосой металла, либо окрашенным профилем.

### Монтажные правила

- Для нижнего ряда применяется начальная планка `ПН`.
- В районе вертикального шва крепежная часть планки подрезается и отгибается вверх.
- `Lp` для `КФ-2` не является шагом вертикальных направляющих.

## КФ-3

### Назначение

- Кассета фасадная типа 3.
- Относится к системе, где вертикальный шов формируется через профиль подсистемы.
- Конструкция позволяет уменьшить горизонтальный и вертикальный швы до `5 мм`, что меняет внешний вид стандартного кассетного фасада.
- Уменьшенная глубина кассеты `17 мм` позволяет использовать стандартные комплектующие элементы для сайдинга и фасадной панели.

### Размеры

- `L` — габаритная ширина.
- `Lp` — монтажная ширина.
- `H` — габаритная высота.
- `Hp` — монтажная высота.
- `H1` — конструктивный размер по высоте.

### Допустимые размеры по СТО

Граничные сочетания:

1. `210 x 360`
2. `580 x 1205`
3. `1205 x 580`

### Стандартная кассета

Стандартная строка:

- `580 x 580`

Правило:

- `КФ-3` стандартная не только как `580 x 580`.
- `КФ-3` стандартная, если хотя бы одна сторона имеет размер `580 мм`.

Примеры:

- `580 x 580` — стандартная
- `580 x 1205` — стандартная
- `1205 x 580` — стандартная
- если ни `L`, ни `H` не равны `580`, кассета не стандартная

### Рекомендации по размерам и жесткости

- Для толщины металла `0,7 мм` рекомендуемые размеры для проектирования — не более `580 x 580 мм`.
- Изготовление кассет больших размеров технически возможно, но нежелательно.
- При увеличенных габаритах, особенно при толщине `0,7 мм`, возможен эффект `бухтения` — вибрация и хлопанье кассеты на ветру.
- Для калькулятора это означает:
- размер больше `580 x 580` не запрещать автоматически
- но выдавать предупреждение по жесткости для `0,7 мм`

### Доборная и угловая

- Если в прайсе есть `доборная`, это доборная кассета.
- Если размеры не содержат `580` ни по `L`, ни по `H`, это нестандартная доборная кассета.
- Если в прайсе есть `угловая`, это угловая кассета.

### Крепеж и отверстия

Подтверждено примечанием к таблице 3:

- центральное отверстие для крепления изготавливается при `L > 799 мм`
- отверстия для удаления влаги:
- `2 отверстия`, если `L <= 799 мм`
- `3 отверстия`, если `L > 800 мм`

Для калькулятора:

- для скрытого крепления `КФ-2` и `КФ-3` принимать `2` самореза на кассету при `L <= 799 мм`;
- при `L > 799 мм` принимать `3` самореза на кассету с учетом центрального крепежного отверстия;
- при `L > 800 мм` в подсистеме дополнительно ставить промежуточные вертикальные профили с шагом не более `600 мм`.

### Швы

- Горизонтальный шов образуется конструктивно и составляет `5 мм`.
- Вертикальный шов формируется произвольно исходя из эстетических соображений:
- от `5 до 20 мм` при монтаже на профиль `ПП 60x27`
- от `5 до 35 мм` при монтаже на шляпный профиль `НПШ20`
- Для обеспечения равномерности шва рекомендуется использовать калиброванный разделитель требуемого размера.

### Подсистема

- Дном вертикального шва является направляющая.
- Направляющая должна быть того же цвета, что и кассета.
- Это решается либо полосой окрашенного металла, закрепленной на направляющей, либо профилем, окрашенным в цвет кассет.
- `Lp` для `КФ-3` не является размером шага вертикальных направляющих.

### Монтажные правила

- Для крепления нижних краев кассеты первого ряда, а также кассет, не имеющих сцепления с нижележащей кассетой, применяется начальный элемент `ПН3`.
- В районе вертикального шва крепежная часть планки подрезается и отгибается вверх.
- Если требуется водоотлив, его устанавливают до монтажа кассет.
- Кассета прикрепляется к вертикальным и горизонтальным направляющим окрашенными самосверлящими винтами `4,8x20` с уплотнительной прокладкой из ЭПДМ-резины диаметром `14 мм`.
- Винты должны устанавливаться в крепежные отверстия строго перпендикулярно лицевой плоскости кассеты.
- При креплении винты нельзя перетягивать.

## КФ-4

### Общая конструктивная логика

- У `КФ-4` нет боковых отгибов слева и справа по схеме `КФ-1`.
- У `КФ-4` есть верхний и нижний горизонтальные отгибы.
- Разницу между `H` и `Hp` можно трактовать как геометрию горизонтального шва.
- Разницу между `L` и `Lp` нельзя механически трактовать по модели `КФ-1`.
- По горизонтальным швам `КФ-4` ближе к `КФ-3`.
- По вертикальной логике `КФ-4` ближе к `КФ-2`.

## КФ-4/30

### Размеры

Стандартная строка таблицы 5:

- `L = 555`
- `H = 572`
- `Lp = 530`
- `Hp = 553`
- `H1 = 525`

### Допустимые размеры по СТО

- минимум: `200 x 407`
- максимум: `1180 x 1197`

### Стандартная кассета

Правило:

- `КФ-4/30` стандартная, если `L = 555` или `H = 572`.

Примеры:

- `555 x 572` — стандартная
- `1180 x 572` — стандартная
- `555 x 1197` — стандартная
- если ни `L`, ни `H` не равны `555` или `572`, кассета не стандартная

### Доборная и угловая

- Если в прайсе есть `доборная`, это доборная кассета.
- Если в прайсе есть `угловая`, это угловая кассета.

### Швы

Горизонтальный шов по геометрии:

- `H = 572`
- `Hp = 553`
- разница `19 мм`

Вывод:

- геометрия горизонтального шва `КФ-4/30` связана с `H - Hp = 19 мм`

Вертикальная логика:

- `L = 555`
- `Lp = 530`
- разница `25 мм`
- эту разницу нельзя трактовать как боковые отгибы по схеме `КФ-1`
- по вертикали `КФ-4/30` требует отдельного монтажного правила

## Что обязательно учитывать в коде

- Стандартность и допустимость размера — это разные проверки.
- Кассета может быть допустимой, но доборной.
- Для каждого типа нужно хранить отдельно:
- допустимые сочетания размеров
- стандартные стороны
- монтажные размеры
- правила швов
- правила крепежа
- связь с подсистемой

## Что по КФ-4 еще требует уточнения

- точная монтажная ширина вертикального шва
- точная логика крепежа
- связь `Lp` с шагом направляющих
- необходимость окрашенной направляющей в шве
- особенности нижнего ряда и стартовых элементов

Пока для `КФ-4` полностью подтверждена геометрическая база по СТО, а часть монтажных правил находится в стадии уточнения.
