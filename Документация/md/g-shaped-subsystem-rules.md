# Г-образная подсистема ИНСИ

Справка по `Г-образной` подсистеме ИНСИ для фасадных кассет, фасадных панелей, металлосайдинга и профлиста.

Документ нужен как инженерная база для калькулятора: что является подтвержденным правилом, что является проектным выбором, и что нельзя автоматически фиксировать без прочностного расчета.

## 1. Что это за система

`Г-образная` подсистема ИНСИ является `вертикально-горизонтальной`.

Логика такая:

1. К стене крепятся `Г-образные` кронштейны `КВГУ`.
2. К выступающей полке кронштейнов крепятся горизонтальные `Г-образные` направляющие `НПГ`.
3. К горизонтальным направляющим крепятся вертикальные несущие профили.
4. К вертикальным профилям крепится облицовка.

Для калькулятора это самостоятельный тип подсистемы. Ее нельзя упрощать до `П-образной`, потому что:

- другой состав;
- другая схема передачи усилий;
- другой набор профилей;
- отдельные ограничения по шагам, зазорам и выносу;
- отдельные решения на наружных углах.

## 2. Состав системы

### 2.1. Обязательные базовые элементы

- `КВГУ` выравнивающий усиленный кронштейн.
- `НПГ` горизонтальная направляющая.
- Вертикальный несущий профиль:
  - `НПШ 20x80x20`;
  - `НПШ 20x50x20`;
  - `Z 40x21x20` для отдельных узлов, в том числе наружных углов.
- Паронитовая терморазрывная прокладка под кронштейн.
- Вытяжные заклепки для соединения элементов подсистемы.
- Анкерный крепеж / дюбель для крепления кронштейна к основанию.
- Шайба усиления кронштейна, если она принята расчетной моделью калькулятора и проектной спецификацией.

### 2.2. Облицовки, которые опираются на систему

Подтверждено применение с:

- фасадными кассетами `КФ1`, `КФ2`, `КФ3`, `КФ4/30`, `КФ4/17`;
- фасадной панелью;
- металлосайдингом;
- профлистом.

## 3. Кронштейны КВГУ

Подтверждены два семейства.

### 3.1. КВГУ 95x85xL

- ширина кронштейна: `95 мм`;
- длина опорной части: `85 мм`;
- длины консоли:
  - `100`
  - `130`
  - `150`
  - `180`
  - `200`
  - `220`
  - `240`
  - `250`
  - `300`
  - `350 мм`
- толщина:
  - `1,5 мм`
  - `2,0 мм`
- особенность: два продольных гофра жесткости высотой `10 мм`.

### 3.2. КВГУ 50x50xL

- ширина кронштейна: `50 мм`;
- длина опорной части: `50 мм`;
- длины консоли:
  - `50`
  - `90`
  - `120`
  - `150`
  - `180`
  - `200`
  - `230 мм`
- по справочным правилам и прайсу встречаются толщины:
  - `1,2 мм`
  - `1,5 мм`
  - `2,0 мм`
- в прочностном заключении уверенно подтверждены `1,5` и `2,0 мм`.

### 3.3. Что важно для калькулятора

Калькулятор должен отдельно хранить:

- тип кронштейна: `КВГУ 95x85` или `КВГУ 50x50`;
- длину консоли;
- толщину;
- покрытие;
- количество.

Калькулятор не должен автоматически считать, что любой вынос можно выполнить любым `КВГУ`: выбор зависит от проекта и прочностного расчета.

## 4. Профили Г-образной системы

### 4.1. Горизонтальные профили

Подтвержденные семейства:

- `НПГ 40x40`
- `НПГ 50x50`
- `НПГ 60x44`
- `НПГ 60x81`

Подтвержденные толщины в сводных правилах:

- `0,9`
- `1,0`
- `1,2`
- `1,5 мм`

В прочностных документах для расчетной схемы явно фигурирует `НПГ 50x50` толщиной:

- `1,2`
- `1,5`
- `2,0 мм`

### 4.2. Вертикальные профили

Подтвержденные семейства:

- `НПШ 20x80x20`
- `НПШ 20x50x20`
- `Z 40x21x20`

Подтвержденные толщины:

- `1,0`
- `1,2`
- `1,5`
- в ряде сводных правил также встречается `0,9 мм`.

### 4.3. Назначение вертикальных профилей

- `НПШ 20x80x20` — основной вертикальный несущий профиль.
- `НПШ 20x50x20` — допустимый тип вертикального профиля, но не базовый по умолчанию для калькулятора, если не введено отдельное правило.
- `Z 40x21x20` — специальный вертикальный профиль, который в документах отдельно упоминается для оформления `наружных углов`.

## 5. Монтажная логика

### 5.1. Крепление к стене

- `КВГУ` крепится к основанию анкерным крепежом или дюбелем.
- Тип и длина анкера определяются расчетом и материалом стены.
- Под кронштейн ставится терморазрывная прокладка.

### 5.2. Паронитовая прокладка

Для `Г-образной` системы подтверждено:

- материал: паронит;
- площадь: `3000 мм²`;
- толщина: `4 мм`.

Для калькулятора это обязательная позиция: `1 прокладка на 1 кронштейн`.

### 5.3. Крепление профилей

- Горизонтальный профиль `НПГ` крепится к кронштейну `двумя` вытяжными заклепками.
- Вертикальный профиль крепится к горизонтальному профилю `двумя` вытяжными заклепками.

Минимальная рабочая модель калькулятора:

- `2 заклепки` на узел `КВГУ -> НПГ`;
- `2 заклепки` на узел `НПГ -> вертикальный профиль`.

## 6. Зазоры и температурные деформации

### 6.1. Горизонтальные профили

Между смежными горизонтальными профилями `НПГ` должен быть температурный зазор:

- `5-15 мм`.

### 6.2. Вертикальные профили

Между смежными вертикальными профилями по высоте должен быть зазор:

- `5-10 мм`.

### 6.3. Общая температурная компенсация

Подтвержденная величина температурных деформаций:

- `1,2-1,5 мм/п.м.`

Это не отдельный “расходный” параметр, а инженерное правило, по которому назначают монтажные зазоры.

## 7. Воздушный зазор и вынос

### 7.1. Минимальный воздушный зазор

Подтверждено:

- воздушный зазор должен быть не менее `40 мм`.

Для калькулятора это жесткая нижняя проверка.

### 7.2. Регулировка относа

Подтверждено:

- собственная регулировка относа горизонтальных направляющих от плоскости выравнивания: до `20 мм`;
- базовый относ кронштейна: `30-40 мм`;
- увеличение относа выполняется подбором длины кронштейна.

### 7.3. Общий вынос облицовки

По сводным правилам и прочностным материалам:

- общий вынос облицовки от базовой стены может доходить до `390-400 мм`.

Это не означает, что любой объект автоматически должен получать максимальный вынос. Для калькулятора это верхняя инженерная граница применимости, а не рекомендуемое значение по умолчанию.

## 8. Шаги элементов

### 8.1. Шаг вертикальных несущих профилей

Подтверждено:

- шаг вертикальных несущих профилей должен быть не более `800 мм`.

Это жесткое правило для калькулятора.

### 8.2. Шаг кронштейнов

По Г-образной системе важно различать:

- `шаг вертикальных профилей` — подтвержден как `не более 800 мм`;
- `шаг кронштейнов` — определяется проектом и прочностным расчетом.

То есть калькулятор не должен выдавать один “нормативный” шаг кронштейнов для всех объектов без оговорки.

Практическое правило для MVP:

- если в интерфейсе вводится шаг кронштейнов, он должен быть подписан как `по расчету / по проекту`;
- без отдельной таблицы несущей способности нельзя выдавать его как универсальную норму.

## 9. Наружные углы

По Г-образной системе наружный угол — не просто фасонный элемент облицовки, а еще и отдельный узел подсистемы.

Подтверждено:

- при оформлении наружных углов применяется вертикальный `Z`-профиль;
- в прочностном заключении отдельно сказано, что `Z 40x21x20` применяется при оформлении внешних углов.

Для калькулятора это означает:

- наружный угол в `Г-образной` системе нельзя считать только “уголком” без учета отдельного профиля подсистемы;
- при угловых кассетах и наружных углах должна быть отдельная логика на вертикальный профиль угла;
- эту часть нужно считать отдельным блоком спецификации подсистемы, а не смешивать с обычными рядовыми вертикальными профилями.

## 10. Связь с типами кассет

### 10.1. КФ1 и КФ4

- крепятся видимым способом;
- опираются на вертикальные направляющие;
- специальных правил по окраске скрытого шва, как у `КФ2`, не требуют.

### 10.2. КФ2 и КФ3

- крепятся скрытым способом;
- видимая часть профиля в вертикальном шве должна быть окрашена в цвет кассет или закрыта декоративной полосой.

Для `Г-образной` системы это правило также важно:

- видимый вертикальный профиль в шве не всегда равен “всей подсистеме в окраске”;
- нужно различать:
  - полностью окрашенную подсистему;
  - оцинкованную подсистему с окрашенным только видимым элементом.

### 10.3. Промежуточный вертикальный профиль

В сводной оцифровке для калькулятора закреплено правило:

- если длина кассеты больше `800 мм`, нужен промежуточный профиль.

Это правило уже можно считать инженерным правилом калькулятора для `Г-образной` системы.

Отдельно важно:

- старые документы по `П-образной` двухуровневой системе местами дают порог `700 мм`;
- для `Г-образной` линии, которую мы берем в калькулятор, уже зафиксировано правило `> 800 мм`.

## 11. Покрытие элементов

Для калькулятора по `Г-образной` системе нужно различать минимум два режима:

### 11.1. Оцинкованная подсистема

- базовые кронштейны и невидимые профили — оцинкованные;
- если кассета скрытого крепления требует видимый профиль в цвет кассеты, он считается отдельно как окрашенный видимый элемент.

### 11.2. Окрашенная подсистема

- кронштейны;
- горизонтальные профили;
- вертикальные профили;
- шайбы, если прайсом предусмотрено окрашенное исполнение;
- видимые элементы шва.

В проектной спецификации такой режим обычно означает окраску элементов системы `с двух сторон`, если прайс содержит именно такую позицию.

## 12. Что можно считать автоматически уже сейчас

Для калькулятора можно уверенно автоматизировать:

1. Выбор схемы `Г-образная`.
2. Проверку минимального воздушного зазора `>= 40 мм`.
3. Подбор семейства кронштейна `КВГУ 50x50` или `КВГУ 95x85`.
4. Подбор длины кронштейна по требуемому выносу.
5. Учет паронитовой прокладки `1:1` к кронштейну.
6. Учет шайбы усиления `1:1` к кронштейну, если принята в модели.
7. Расчет длины горизонтальных профилей.
8. Расчет длины вертикальных профилей.
9. Проверку, что шаг вертикальных профилей не превышает `800 мм`.
10. Добавление промежуточного вертикального профиля при длине кассеты `> 800 мм`.
11. Разделение видимых и невидимых окрашиваемых элементов для `КФ2/КФ3`.
12. Учет отдельного углового вертикального профиля на наружных углах.

## 13. Что нельзя жестко фиксировать без доп. расчета

Нельзя автоматически объявлять универсальной нормой:

- точный шаг кронштейнов;
- несущую способность кронштейна по любому выносу;
- допустимость замены `КВГУ 95x85` на `КВГУ 50x50` только по геометрии;
- допустимость конкретной толщины профиля без проверки ветровой нагрузки и схемы опирания;
- расчет анкеров по материалу стены.

Это должно идти как:

- проектный параметр;
- либо как следующий этап развития калькулятора с таблицами несущей способности.

## 14. Рекомендация для модели калькулятора

Для `Г-образной` подсистемы модель должна хранить:

- `bracketFamily`
- `bracketLength`
- `bracketThickness`
- `bracketCoating`
- `airGap`
- `requiredOffset`
- `horizontalProfileFamily`
- `horizontalProfileThickness`
- `verticalProfileFamily`
- `verticalProfileThickness`
- `verticalProfilePitch`
- `thermalGapHorizontal`
- `thermalGapVertical`
- `requiresIntermediateVerticalProfile`
- `requiresVisibleColoredProfile`
- `requiresOuterCornerVerticalProfile`

## 15. Рабочие правила для следующего этапа внедрения в интерфейс

Если переводить эту справку в логику калькулятора, то базовая схема для `Г-образной` системы должна быть такой:

1. Пользователь задает воздушный зазор и тип исполнения подсистемы.
2. Калькулятор подбирает семейство и длину `КВГУ`.
3. Горизонтальный профиль считается по рядам кронштейнов.
4. Вертикальный профиль считается по линиям раскладки кассет.
5. Если кассета длиннее `800 мм`, добавляется промежуточный вертикальный профиль.
6. Если есть наружные углы, добавляется угловой вертикальный профиль `Z`.
7. Если кассета `КФ2` или `КФ3`, видимый профиль в шве считается как окрашенный элемент.

## 16. Короткий вывод

`Г-образная` подсистема ИНСИ — это не “упрощенная версия П-образной”, а отдельная вертикально-горизонтальная система со своим набором:

- `КВГУ`;
- `НПГ`;
- `НПШ / Z`;
- температурных зазоров;
- угловых узлов;
- правил по видимым профилям.

Для калькулятора уже есть достаточная подтвержденная база, чтобы:

- сделать отдельную спецификацию `Г-образной` системы;
- корректно выбирать `КВГУ`;
- считать `НПГ` и вертикальные профили;
- учитывать наружные углы;
- отдельно считать окрашенные видимые элементы для скрытого крепления кассет.

Но точная несущая модель по шагам и толщине должна оставаться связанной с проектным или прочностным расчетом.
