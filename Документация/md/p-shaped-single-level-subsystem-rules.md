# П-образная одноуровневая подсистема для фасадных кассет

Документ описывает состав, нормы и рабочие правила расчета
`П-образной одноуровневой подсистемы` для калькулятора фасадных кассет.

Это справочный документ для реализации калькулятора. Он отделяет
подтвержденные требования из документации от расчетных допущений MVP.

## 2. Назначение системы

`П-образная одноуровневая система` применяется как выравнивающая система
для крепления фасадных кассет к стене здания.

В документации указано, что одноуровневая система для кассет по логике
повторяет вертикальную выравнивающую систему.

Практический смысл:

- на стену устанавливаются П-образные кронштейны;
- на кронштейны крепятся вертикальные П-образные направляющие;
- кассеты крепятся к вертикальным направляющим;
- система компенсирует неровности основания и формирует плоскость
  облицовки.

## 4. Толщины элементов

### Кронштейны

В документации по П-образной системе упоминаются:

- `КВП 200`;
- `КВП 250`;
- толщина кронштейна `1,0 мм`.

В прайсе толщина в наименовании `КВП 200/250` явно не всегда вынесена в
название, поэтому для калькулятора типоразмер кронштейна хранится как
прайсовая позиция.

### Направляющие

Для П-образной подсистемы подтвержден профиль:

- `НПП 60x27`;
- толщина `1,0 мм`.

В прайсе также есть другие толщины `0,5 / 0,7 / 1,0 мм`, но для
одноуровневой расчетной системы в калькуляторе базово использовать
`НПП 60x27x1,0`.

### Прокладки

Для паронитовой прокладки подтверждены:

- площадь `3000 мм²`;
- толщина `4 мм`.

Цена прокладки в текущем прайсовом справочнике не подтверждена.

## 5. Нормы и ограничения

### Воздушный зазор

Минимальный вентиляционный зазор:

```text
не менее 40 мм
```

В документации встречается диапазон:

```text
40-100 мм
```

Для калькулятора:

- не позволять принимать воздушный зазор меньше `40 мм`;
- значения выше `100 мм` требуют проверки по проекту и выбору кронштейна.

### Шаг направляющих

В монтажной инструкции указано, что шаг направляющих определяется расчетом.

В расчетном примере одноуровневой системы встречается:

```text
шаг направляющих b = 800 мм
```

Для калькулятора:

- шаг вертикальных линий должен быть связан с раскладкой кассет;
- если расчет ведется без раскладки кассет, принимать не более `800 мм`;
- для кассет вертикальные линии обычно привязаны к вертикальным швам и
  краям кассетного поля.

### Шаг кронштейнов

В монтажной инструкции указано:

```text
шаг крепления кронштейнов определяется расчетом
```

В расчетном примере одноуровневой системы встречается:

```text
шаг кронштейнов L = 800 мм
```

В других расчетных фрагментах встречается `1000 мм`, но это пример, а не
универсальная норма.

Для калькулятора MVP принято консервативное правило:

```text
максимальный шаг кронштейнов = 800 мм
```

Пользователь может задавать меньший шаг. Значение больше `800 мм` не
принимается без отдельного проектного/прочностного расчета.

### Температурные деформации

Подтвержденное правило:

```text
1,2-1,5 мм/п.м.
```

Между смежными вертикальными направляющими требуется компенсационный
зазор:

```text
5-10 мм
```

В документации также встречается ориентир около `10 мм`.

### Запрет жесткой стыковки

Вертикальные направляющие нельзя жестко скреплять между собой в одну
непрерывную линию.

Стыки должны оставаться компенсационными.

## 6. Стыковка профилей длиной 3 м

В документации указано, что длина направляющих ИНСИ составляет `3 м`.

При высоте фасада больше `3000 мм` вертикальная линия собирается из
нескольких профилей.

Правила:

- профиль стыкуется по вертикали с компенсационным зазором;
- зазор между смежными профилями `5-10 мм`;
- стык нельзя жестко соединять;
- при кассетной облицовке желательно совмещать стык профиля с
  горизонтальным стыком кассет;
- при расчете закупки можно считать хлысты `3000 мм`, но в монтажной
  схеме нужно показывать стыки и зазоры.

### Расчет количества хлыстов

Для предварительной спецификации:

```text
profilePcs = ceil(totalProfileLm / 3)
```

Для более точной монтажной схемы:

```text
profilePcsPerLine = ceil(facadeHeight / 3000)
totalProfilePcs = profilePcsPerLine * verticalLines * facadeQuantity
```

Второй способ лучше для монтажной схемы, потому что не смешивает остатки
профилей между разными вертикальными линиями.

## 7. Правила расчета для калькулятора

### 7.1. Вертикальные линии подсистемы

Если уже рассчитана раскладка кассет, вертикальные линии подсистемы
привязываются к кассетной сетке.

Базовое правило:

```text
verticalLines = columns + 1
```

где `columns` — количество кассет/позиций по длине фасада.

Если используются угловые кассеты, количество рядовых кассет по длине
берется внутри поля между полками угловых кассет.

Если раскладки кассет нет:

```text
verticalLines = ceil(facadeWidth / 800) + 1
```

### 7.2. Ряды кронштейнов по высоте

Для фасада высотой `Hf` и шага кронштейнов `S`:

```text
bracketRows = ceil(Hf / S) + 1
```

Минимально:

```text
bracketRows >= 2
```

Для MVP:

```text
S <= 800 мм
```

### 7.3. Количество кронштейнов

Для одного типа фасада:

```text
bracketQty = verticalLines * bracketRows * facadeQuantity
```

### 7.4. Паронитовые прокладки

На каждый кронштейн:

```text
paroniteGasketQty = bracketQty
```

### 7.5. Шайбы усиления кронштейна

В рабочей модели калькулятора шайба усиления считается на каждый
кронштейн:

```text
bracketWasherQty = bracketQty
```

Если в будущем проектная схема будет разрешать применять шайбу только в
отдельных зонах, нужно добавить переключатель или коэффициент.

### 7.6. Анкеры / фасадные дюбели

На каждый кронштейн минимум один крепеж к основанию:

```text
anchorQty = bracketQty
```

Пока отдельная прайсовая позиция анкера не заведена в справочник, строка
может иметь статус `needs_price`.

### 7.7. Заклепки

Подтверждено, что вертикальные `НПП` направляющие крепятся к кронштейнам
стальными вытяжными заклепками `4,0-5,0 мм`.

Рабочее правило MVP:

```text
rivetQty = bracketQty * 2
```

Если в узле применяется две заклепки с каждой стороны или дополнительные
крепления, коэффициент должен быть изменен.

Важно: ранее в интерфейсе временно использовалось `4` заклепки на
кронштейн для общего расчета подсистемы. Для П-образной одноуровневой
системы правильнее начать с отдельного коэффициента `2` и дать его
настраивать.

### 7.8. Вертикальный профиль НПП

Погонные метры:

```text
nppLm = verticalLines * facadeHeight / 1000 * facadeQuantity
```

Количество 3-метровых хлыстов:

```text
nppPcs = ceil(facadeHeight / 3000) * verticalLines * facadeQuantity
```

Для спецификации можно показывать оба значения:

- `п.м` для цены, если прайс в погонных метрах;
- `шт по 3 м` для монтажной логики.

## 8. Окрашенное / оцинкованное исполнение

### Проектная спецификация

Для проектного режима в калькуляторе допускается принимать подсистему в
окрашенном исполнении, если это требуется эстетикой шва.

### Монтажная схема

Для монтажной схемы оцинкованная подсистема допустима, кроме видимых
частей.

### КФ-2 и КФ-3

Для `КФ-2` и `КФ-3` вертикальная направляющая, которая формирует дно
вертикального шва, должна быть:

- окрашена в цвет кассет;
- или закрыта декоративной полосой из материала кассет.

Это обязательно учитывать в спецификации, потому что видимая часть
подсистемы влияет на внешний вид фасада.

## 9. Что входит в спецификацию

Для П-образной одноуровневой системы в спецификацию должны попадать:

1. `КВП` кронштейн.
2. `НПП 60x27x1,0` вертикальная направляющая.
3. Паронитовая прокладка.
4. Шайба усиления кронштейна `Ш50`.
5. Анкер / фасадный дюбель.
6. Заклепки сталь/сталь.

Дополнительно, если выбран вариант декоративной полосы:

7. Декоративная полоса в цвет кассет.

## 10. Что не относится к одноуровневой П-системе

В базовую одноуровневую систему не входят:

- сплошной горизонтальный уровень `НПП`;
- второй уровень подсистемы;
- горизонтальные направляющие по всей плоскости фасада;
- шляпный профиль `НПШ` как отдельный второй уровень.

Эти элементы относятся к двухуровневой системе или специальным узлам.

## 11. Контрольный пример

Дано:

```text
Фасад: 12000 x 6500 мм
Количество: 1
Шаг кронштейнов: 800 мм
Вертикальные линии: 21
```

Ряды кронштейнов:

```text
ceil(6500 / 800) + 1 = 10
```

Кронштейны:

```text
21 * 10 * 1 = 210 шт
```

Паронитовые прокладки:

```text
210 шт
```

Шайбы усиления:

```text
210 шт
```

Заклепки при коэффициенте `2`:

```text
210 * 2 = 420 шт
```

Погонные метры `НПП`:

```text
21 * 6500 / 1000 = 136,5 п.м
```

3-метровые профили:

```text
ceil(6500 / 3000) * 21 = 3 * 21 = 63 шт
```

## 12. Ограничения и вопросы для уточнения

- Точный шаг кронштейнов должен определяться расчетом по нагрузкам,
  основанию, высоте здания и ветровому району.
- Максимум `800 мм` принят в калькуляторе как консервативное рабочее
  ограничение MVP.
- Нужно добавить отдельный выбор `КВП 200 / КВП 250`.
- Нужно добавить отдельный коэффициент заклепок на кронштейн.
- Нужно добавить строку анкеров/дюбелей после подтверждения прайсовых
  позиций.
- Нужно уточнить, когда шайба усиления обязательна всегда, а когда только
  по расчету.
- Нужно отдельно описать узлы проемов, углов и примыканий.
