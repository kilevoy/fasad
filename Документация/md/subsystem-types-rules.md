## Общая логика

Система `ИНСИ СК` для металлических облицовок не сводится к одному типу подсистемы.
Для калькулятора нужно поддерживать минимум три большие ветки:

1. `стандарт` с креплением к стене;
2. `каркас` с креплением к межэтажным перекрытиям;
3. внутри `стандарт` отдельно считать:
- `П-образную` систему;
- `Г-образную` систему.

## Исполнение `стандарт`

Исполнение `стандарт` используется для крепления системы к стенам здания.

В нем есть два основных конструктивных варианта:
- `П-образная система`;
- `Г-образная система`.

## П-образная подсистема

Подробная справка по одноуровневой П-образной системе для калькулятора:
`p-shaped-single-level-subsystem-rules.md`.

### Состав

Несущий каркас `П-образной` системы включает:
- выравнивающие кронштейны `КВП 200`, `КВП 250`, толщина `1,0 мм`;
- несущие вертикальные и горизонтальные направляющие `НПП 60x27`, толщина `1,0 мм`;
- основные несущие вертикальные профили для кассет `НПШ 20x80`, толщина `0,9 / 1,0 / 1,2 мм`;
- промежуточные несущие вертикальные профили `НПШ 20x50`, толщина `0,9 / 1,0 / 1,2 мм`.

### Что умеет кронштейн КВП

- регулирует относ направляющих от стены;
- позволяет компенсировать отклонения стены от вертикали;
- базово дает относ `30-40 мм`;
- больший относ достигается подбором длины кронштейна.

### Крепление П-образной системы

- к стене: распорные дюбели с металлическими шурупами или химические анкеры;
- к металлическому каркасу: болтовые или заклепочные соединения;
- под кронштейны ставятся терморазрывные прокладки.
- при усилении узла учитывать шайбу усиления кронштейна `Ш50`, если она предусмотрена схемой.

## Варианты П-образной системы

### 1. Вертикальная выравнивающая система

Используется для горизонтального монтажа облицовки:
- металлосайдинг;
- фасадные панели;
- профлист.

Правила:
- вертикальные направляющие `НПП` крепятся к кронштейнам стальными вытяжными заклепками `4,0-4,8 мм`;
- оставшиеся консольные части кронштейнов после крепления отгибаются внутрь подсистемы;
- сопряжение направляющих делается с температурным зазором;
- компенсация температурных деформаций: `1,2-1,5 мм/п.м.`;
- рабочий зазор между смежными направляющими: `5-10 мм`;
- запрещено скреплять вертикальные направляющие между собой.

### 2. Горизонтальная выравнивающая система

Используется для вертикального монтажа облицовки.

Правила:
- вертикальные `НПП` направляющие крепятся к кронштейнам;
- между вертикальными направляющими ставятся дополнительные горизонтальные `НПП` профили;
- соединение вертикальных и горизонтальных направляющих выполняется `внахлест`;
- у горизонтальных направляющих подрезаются боковые полки;
- крепление выполняется стальными вытяжными заклепками.

### 3. Одноуровневая система для кассет

По логике повторяет вертикальную выравнивающую систему.

Правила:
- вертикальные направляющие `НПП` крепятся к кронштейнам заклепками `4,0-5,0 мм`;
- температурные деформации учитываются так же: `1,2-1,5 мм/п.м.`;
- компенсационный зазор: `5-10 мм`;
- скреплять вертикальные направляющие между собой нельзя.

Особенности по кассетам:
- для `КФ2` используют вертикальные направляющие, окрашенные в цвет кассет;
- либо на видимую часть профиля ставится декоративная полоса из материала кассет.

### 4. Двухуровневая система для кассет

Состав:
- вертикальные направляющие `НПП`;
- дополнительные горизонтальные `НПП`;
- вертикальный `НПШ` профиль, закрепленный на горизонтальных направляющих.

Правила:
- `НПШ` ставится с шагом, равным монтажному размеру кассет;
- если длина кассеты более `800 мм`, в средней части кассеты ставится промежуточный `ПШ` профиль;
- для `КФ2` основные `НПШ` направляющие должны быть окрашены в цвет кассет либо закрыты декоративной полосой.

Рабочее решение для калькулятора:

- основной вертикальный профиль считать как `НПШ 20x80x20 1,0`;
- промежуточный вертикальный профиль в калькуляторе также считать как `НПШ 20x80x20 1,0`;
- отдельный переход на `НПШ 20x50x20` пока не использовать, пока не будет отдельного проектного правила.

## Г-образная подсистема

Подробная справка по `Г-образной` системе для калькулятора:
`g-shaped-subsystem-rules.md`.

Отдельная сводная справка по организации подсистемы на углах:
`subsystem-corners-rules.md`.

### Общая схема

`Г-образная` система является `вертикально-горизонтальной`.

Для калькулятора это отдельный тип подсистемы, который нельзя подменять `П-образной`.

### Кронштейны

Подтверждены два типа:

1. `КВГУ 95x85`
- длина консоли `100 / 130 / 150 / 180 / 200 / 220 / 240 / 250 / 300 / 350 мм`;
- толщина `1,5` и `2,0 мм`;
- два продольных гофра жесткости высотой `10 мм`.

2. `КВГУ 50x50`
- длины `50 / 90 / 120 / 150 / 180 / 200 / 230 мм`;
- в пояснительной записке упомянут как `КВГУ50x50`;
- в прочностном заключении фигурирует как `КВГУ 50x50xL`;
- по таблицам встречаются толщины `1,2 / 1,5 / 2,0 мм`.

### Направляющие Г-образной системы

Горизонтальные:
- `НПГ 40x40`;
- `НПГ 50x50`;
- `НПГ 60x44`;
- `НПГ 60x81`;
- толщины `0,9 / 1,0 / 1,2 / 1,5 мм`.

Вертикальные:
- `НПШ 80x20`;
- `НПШ 50x20`;
- `Z 40x20`;
- толщины `0,9 / 1,0 / 1,2 / 1,5 мм`.

### Монтажная логика Г-образной системы

- кронштейн крепится к стене анкерным крепежом;
- под кронштейн ставится терморазрывная прокладка из паронита;
- площадь паронитовой прокладки `3000 мм²`, толщина `4 мм`;
- размер относа горизонтальных направляющих от плоскости выравнивания регулируется до `20 мм`;
- минимальный воздушный зазор должен быть не менее `40 мм`;
- тип кронштейна выбирается по толщине утеплителя, размеру воздушного зазора и фактическому отклонению стены;
- разные типоразмеры кронштейнов позволяют вынести облицовку от стены до `390-400 мм`.

### Соединения

- к выступающей полке кронштейна горизонтальный `Г`-образный профиль крепится двумя вытяжными заклепками;
- между смежными горизонтальными профилями устраивается температурный зазор `5-15 мм`;
- к горизонтальным профилям двумя вытяжными заклепками крепится вертикальный `НПШ` профиль;
- вертикальные несущие профили должны ставиться с шагом не более `800 мм`;
- между смежными вертикальными профилями по высоте устраивается зазор `5-10 мм`.

### Что особенно важно для калькулятора

Для `Г-образной` подсистемы нужно уметь считать:
- тип кронштейна;
- длину консоли кронштейна;
- толщину кронштейна;
- количество паронитовых прокладок;
- количество шайб усиления кронштейна;
- тип горизонтального профиля;
- тип вертикального профиля;
- шаг вертикальных профилей;
- температурные зазоры;
- допустимый относ;
- минимальный воздушный зазор.

## Исполнение `каркас`

Это отдельная подсистема для крепления к торцам межэтажных перекрытий каркасных зданий.

### Состав

- парные кронштейны `КВГУ 95x85`;
- вертикальные профили `ПГС 103x50(60)x13`, толщина `1,2 / 1,5 / 2,0 мм`;
- соединительные элементы `ССП`;
- скоба соединительная `СС`, толщина `1,0 мм`.

### Правила

- крепление к торцу перекрытия: двумя распорными или химическими анкерами;
- воздушный зазор не менее `40 мм`;
- вынос облицовки от базовой стены до `380 мм`;
- вертикальные профили крепятся к кронштейнам вытяжными заклепками `4,8x8`;
- между торцами смежных направляющих оставляют компенсационный зазор `5-10 мм`;
- `ССП` крепится только к нижней направляющей;
- горизонтальный шаг кронштейнов определяется проектом;
- вертикальный шаг задается высотой этажа.

### Проемы

Для окон и дверей:
- применяются горизонтальные `С`-образные профили;
- вертикальная обвязка может выполняться `П-образными 60x27x1,0` или `Г-образными 50x50x1,0`;
- выбор профиля зависит от размеров проема и прочностного расчета;
- при ширине окна более `1200 мм` ставятся промежуточные вертикальные `ПГС` профили.

Подробные правила расчета рамки подсистемы, откосов, аквилонов, водоотливов и утеплительной окантовки вынесены в:
- [opening-subsystem-rules.md](C:\КАЛЬКУЛЯТОР КАССЕТ\Документация\md\opening-subsystem-rules.md)

## Облицовка и привязка к подсистеме

Подтвержденные облицовки:
- металлосайдинг `МС`;
- фасадные панели `ПФ`;
- фасадные кассеты `КФ1`, `КФ2`, `КФ3`, `КФ4/30`, `КФ4/17`;
- профилированный лист.

Связи с подсистемой:
- `КФ2` и `КФ3` требуют невидимого крепления;
- `КФ2` требует окрашенных в цвет кассет видимых направляющих или декоративной полосы;
- `КФ1` и `КФ4` крепятся видимым способом к вертикальным направляющим;
- при длине кассеты более `800 мм` в двухуровневой системе ставится промежуточный профиль.

## Крепеж

Подтверждено:
- заклепки для крепления направляющих: `4,0-4,8 мм`;
- для одноуровневой и двухуровневой системы встречается `4,0-5,0 мм`;
- начальные планки `ПНс` и `ПН` крепятся самосверлящими винтами `4,2x16 мм`;
- `КФ2` и `КФ3` крепятся оцинкованными самосверлящими винтами с ЭПДМ, диаметр `4,8 мм`;
- часть фасонных изделий крепится винтами с прессшайбой `4,2 мм`;
- `КФ1` и `КФ4` крепятся оцинкованными саморезами с окрашенной шестигранной головкой и ЭПДМ.

## Утепление

Правила для калькулятора:
- толщина утеплителя определяется теплотехническим расчетом;
- допустимы негорючие минераловатные плиты;
- при однослойном утеплении плотность не менее `75 кг/м3`;
- при большой толщине утепление делится на 2 слоя;
- для внутреннего слоя допускается плотность не менее `30 кг/м3`, если материал разрешен для вентфасадов;
- стекловолокнистый утеплитель допустим только в комбинации с наружным минераловатным слоем не менее `30 мм`;
- вокруг окон и дверей нужна окантовка из минераловатных плит шириной не менее `150 мм`.

### Дюбели

- однослойное утепление: не менее `5 шт/м2`;
- крайние и угловые зоны: `6 шт/м2`;
- двухслойное утепление: не менее `10 шт/м2`;
- крайние и угловые зоны при двухслойном: `12 шт/м2`;
- для первого слоя при плотности от `30 кг/м3` тарелка дюбеля не менее `150 мм`.

## Что важно заложить в модель калькулятора

### Для подсистемы

- `systemExecution`: `standard | frame`
- `subsystemShape`: `p_shaped | g_shaped`
- `levelingDirection`: `vertical | horizontal | vertical_horizontal`
- `cassetteMountLevel`: `single_level | double_level`
- `bracketType`
- `bracketLength`
- `bracketThickness`
- `horizontalProfileType`
- `horizontalProfileThickness`
- `verticalProfileType`
- `verticalProfileThickness`
- `decorativeStripRequired`
- `airGap`
- `thermalGapBetweenGuides`

### Для инженерных ограничений

- минимальный воздушный зазор `40 мм`
- шаг вертикальных профилей `не более 800 мм` для `Г-образной` системы
- температурные зазоры:
  - `5-10 мм` между направляющими;
  - `5-15 мм` между горизонтальными профилями `Г`-системы
- промежуточный профиль при длине кассеты `> 800 мм`

## Что уже можно считать в калькуляторе

1. Выбор между `П-образной`, `Г-образной` и системой `каркас`.
2. Подбор кронштейна по типу, длине и толщине.
3. Подбор профилей по типу системы.
4. Расчет воздушного зазора.
5. Расчет утеплителя и дюбелей.
6. Нужность декоративной полосы или окрашенного профиля для `КФ2`.
7. Добавление промежуточного профиля при кассетах длиной более `800 мм`.
