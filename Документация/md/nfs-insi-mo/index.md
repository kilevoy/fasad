# НФС ИНСИ МО

## Статус набора

Этот каталог хранит рабочую оцифровку и извлеченные правила по документам из набора `НФС ИНСИ МО`.

Цель:
- собрать правила для калькулятора подсистемы и облицовки;
- отделить действительно расчетные документы от сертификатных и согласовательных;
- зафиксировать, какие файлы уже разобраны, а какие еще требуют тяжелого OCR.

## Состав исходников

Источник: [Документация\Исходники\НФС ИНСИ МО](C:\КАЛЬКУЛЯТОР КАССЕТ\Документация\Исходники\НФС ИНСИ МО)

### Уже просмотрены и полезны для калькулятора

1. `ПЗ к ТРСФИ 1-2023-1.docx`
- Пояснительная записка к альбому технических решений.
- Самый полезный документ для архитектуры калькулятора подсистемы.
- Содержит описание вариантов системы, элементов, крепежа, утепления и монтажа.

2. `АТР 1-2023 МО без ПЗ.pdf`
- Альбом технических решений.
- Полезен для узлов, схем выравнивающей системы, окон, углов, цоколя, парапета, крепления в перекрытие.
- Текст в PDF читается частично, но для узлов документ очень ценный.

3. `Э.З. прочностной НФС с МО 2023.pdf`
- Экспертное заключение по несущей способности элементов каркаса.
- Ключевой документ для типов подсистем, диапазонов профилей, кронштейнов, минимальных толщин и расчетной логики.

### Полезны, но пока не оцифрованы полностью

4. `Заключение прочн расчет Г образная подсистема.jpg.pdf`
- Похоже на отдельный материал по расчету `Г-образной` подсистемы.
- Важен для подсистемы, но файл скановый и требует отдельного OCR-прохода.

5. `ЭЗ пожарка 2023.pdf`
- Пожарное заключение.
- Важно для допусков применения системы, но меньше влияет на количественный расчет материалов.
- Нужен для блока ограничений и применимости, а не для базовой формулы калькулятора.

6. `ТС на фасады с метал. облицовками .pdf`
- Техническое свидетельство.
- Важно для нормативной валидности системы и правил применения.
- Для ядра количественного расчета вторично, но для карточки системы и ограничений полезно.

## Практический вывод

Для разработки калькулятора уже достаточно материала, чтобы:
- формализовать типы подсистем;
- разделить `П-образную`, `Г-образную` и систему с креплением в перекрытие;
- заложить типы кронштейнов, направляющих, шаги, зазоры, крепеж и утепление;
- связать выбор подсистемы с типом облицовки и способом монтажа.
