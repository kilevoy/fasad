# Проемы: Подсистема И Обрамление

Справка для калькулятора по тому, как учитывать оконные и дверные проемы в подсистеме, облицовке, фасонных элементах и утеплении.

## 1. Общий принцип

Проем в калькуляторе нельзя учитывать только как вычет площади фасада. Для профессиональной спецификации проем влияет сразу на несколько блоков:

- площадь рядовых кассет;
- раскладку кассет вокруг проема;
- дополнительные профили подсистемы вокруг проема;
- фасонные элементы откосов, верхнего и нижнего примыкания;
- утеплитель и окантовку проема;
- крепеж фасонных элементов и профилей.

Рабочая логика: сначала считаем геометрию проема, потом строим рамку подсистемы вокруг него, потом выбираем узел обрамления откоса.

## 2. Подсистема вокруг проема

По документации для оконных и дверных проемов применяются дополнительные элементы обрамления.

Подтвержденные правила:

- вокруг окна строится отдельная рамка профилей;
- для вертикальной выравнивающей системы показана схема обрамления окна;
- для горизонтальной выравнивающей системы также показана отдельная схема;
- габарит рамки по ширине в схеме: `L окна + 200 мм`;
- боковые выносы от проема в схеме: по `100 мм` с каждой стороны;
- при ширине окна более `1200 мм` ставятся промежуточные вертикальные `ПГС` профили;
- для проемов применяются горизонтальные `С`-образные профили;
- вертикальная обвязка может выполняться `П-образными 60x27x1,0` или `Г-образными 50x50x1,0`;
- выбор профиля зависит от размера проема и прочностного расчета.

## 3. Что считать в подсистеме по каждому проему

Для калькулятора по каждому проему нужно считать отдельный комплект:

- вертикальные профили обрамления слева и справа;
- горизонтальные профили сверху и снизу;
- промежуточные вертикальные профили при ширине проема более `1200 мм`;
- крепеж профилей к кронштейнам или к основной подсистеме;
- кронштейны, если профиль обрамления требует самостоятельного крепления к основанию;
- паронитовые прокладки под кронштейны, если используются кронштейны;
- крепеж фасонных элементов откоса к несущим профилям.

Базовая расчетная рамка:

- расчетная ширина рамки: `openingWidth + 200 мм`;
- расчетная высота рамки: `openingHeight + 200 мм`, если нет отдельного узла с другими отступами;
- вертикальные профили рамки: `2 * расчетная высота`;
- горизонтальные профили рамки: `2 * расчетная ширина`;
- промежуточные вертикальные профили: если `openingWidth > 1200 мм`, количество и шаг задаются расчетом, на MVP можно принять `ceil(openingWidth / 1200) - 1`.

Важно: это рабочее правило для калькулятора. Окончательный профиль и количество кронштейнов должны уточняться по прочностному расчету и выбранной системе.

## 4. Откосы и фасонные элементы

Документы подтверждают, что для оконных и дверных откосов применяются фасонные комплектующие изделия.

Подтвержденные элементы:

- `Аквилон (Акс)` — приемный профиль узла откоса, применяется в паре с откосным элементом;
- `Откосная планка (ОПс 180/250)` — элемент обрамления откоса;
- `Откос декоративный (ОДс)` — декоративное оформление откоса;
- `Уголок откосный (УО)` — обрамление откосов окон и верхнего откоса окна;
- `Водоотлив (Вс/ВО)` — защита нижнего откоса окна от осадков;
- `Отлив верхний (ОВс)` — верхнее примыкание;
- `Наличник (НЛс)` — обрамляющий элемент проема, если выбран соответствующий узел.

Подтвержденные схемы:

- `Аквилон + откосная планка`;
- `Аквилон + откос декоративный`;
- `Аквилон + уголок откосный`;
- нижний узел: `Водоотлив`;
- верхний узел: `Отлив верхний` или другой элемент по узлу проекта.

Ключевой вывод: `Аквилон` не заменяет `уголок откосный`; он работает как приемный профиль в составе узла.

### 4.1. Учет выноса системы для откосов и водоотливов

Для калькулятора откосный элемент и водоотлив нельзя выбирать только по ширине проема. Нужно учитывать глубину от базовой плоскости проема до наружной плоскости облицовки.

Рабочая формула:

`глубина откоса = толщина утеплителя + вентзазор + глубина несущего профиля + толщина кассеты + запас на подгиб/посадку`

Принято в текущей версии:

- для П-образной одноуровневой системы глубина профиля: `27 мм`;
- для П-образной двухуровневой системы глубина шляпного профиля: `20 мм`;
- для Г-образной системы глубина Г-профиля: `20 мм`;
- запас на подгиб/посадку: `30 мм`.

Выбор типоразмера:

| Элемент | Условие | Позиция |
| --- | --- | --- |
| Откосный элемент | глубина `<= 190 мм` | `ОЭк190` |
| Откосный элемент | глубина `> 190 мм` и `<= 260 мм` | `ОЭк260` |
| Откосный элемент | глубина `> 260 мм` | индивидуальный доборный элемент / требуется проверка |
| Водоотлив | глубина `<= 180 мм` | `Вс180` |
| Водоотлив | глубина `> 180 мм` и `<= 250 мм` | `Вс250` |
| Водоотлив | глубина `> 250 мм` | индивидуальный доборный элемент / требуется проверка |

Длина элементов по фронту проема пока считается по ширине/высоте проема. Боковые выпуски водоотлива и дополнительные завороты нужно добавить отдельным правилом, если они будут утверждены.

## 5. Крепление элементов проема

Подтверждено:

- часть фасонных изделий, включая откосные планки и наличники, крепится к несущим профилям;
- для откосных планок указан крепеж `4,2x16`;
- локально в узлах встречаются заклепки `4,8x8` и самосверлящие винты `4,2x16`;
- точный шаг крепления фасонных элементов в доступных документах не найден.

Для калькулятора:

- тип крепежа можно заложить по группе элемента;
- шаг крепления хранить как параметр справочника `requires_confirmation`;
- количество крепежа считать только после выбора шага, либо выводить как расчетную позицию с пометкой "требует подтверждения".

## 6. Утеплитель вокруг проема

Подтверждено:

- вокруг окон и дверей требуется окантовка из минераловатных плит шириной не менее `150 мм`.

Для калькулятора:

- площадь основного утеплителя можно считать за вычетом площади проемов;
- отдельно считать дополнительную минераловатную окантовку;
- базовая длина окантовки по одному проему: `2 * (openingWidth + openingHeight)`;
- базовая площадь полосы окантовки: `perimeter * 150 мм`;
- если окантовка идет отдельной плотностью/типом плиты, выводить отдельной строкой спецификации.

## 7. Влияние проемов на кассеты

Проем влияет на кассеты не только как вычет площади.

Нужно учитывать:

- кассеты вокруг проема могут становиться доборными;
- раскладка по фасаду должна разбиваться на зоны: слева от проема, над проемом, справа от проема, под проемом;
- если проем попадает в поле рядовых кассет, нельзя просто вычесть площадь окна из площади кассет: нужно пересчитать раскладку;
- вокруг проема появляются откосные и примыкающие элементы, которые входят в отдельную спецификацию.

Для MVP можно считать:

- геометрическую площадь кассет `площадь фасада - площадь проемов`;
- фасонные элементы по периметру проема;
- подсистему вокруг проема отдельной рамкой.

Для профессиональной версии нужно:

- хранить положение проема на фасаде;
- строить раскладку кассет с учетом координат проема;
- выделять доборные кассеты по сторонам проема;
- отдельно считать верхние и нижние доборные ряды.

## 8. Минимальная модель данных для калькулятора

По каждому проему нужны поля:

- `type`: окно, дверь, ворота, произвольный;
- `widthMm`;
- `heightMm`;
- `quantity`;
- `xMm` и `yMm`, если включена точная раскладка;
- `revealDepthMm` или расчет от выноса подсистемы;
- `trimScheme`: схема обрамления;
- `hasDrip`: нужен ли водоотлив;
- `hasTopFlashing`: нужен ли верхний отлив;
- `subsystemFrameMode`: автоматическая рамка или ручной узел;
- `insulationBorderWidthMm`: по умолчанию `150`.

## 9. Рабочие формулы для первой реализации

### Подсистема проема

```
frameWidthMm = openingWidthMm + 200
frameHeightMm = openingHeightMm + 200
verticalProfileLm = 2 * frameHeightMm / 1000
horizontalProfileLm = 2 * frameWidthMm / 1000
intermediateVerticalProfiles = openingWidthMm > 1200 ? ceil(openingWidthMm / 1200) - 1 : 0
intermediateVerticalLm = intermediateVerticalProfiles * frameHeightMm / 1000
```

Профили переводить в складскую длину `3 м` с округлением вверх, если используется стандартный профиль 3 м.

### Фасонные элементы

```
sideRevealLm = 2 * openingHeightMm / 1000
topRevealLm = openingWidthMm / 1000
bottomDripLm = openingWidthMm / 1000
aquilonLm = sideRevealLm + topRevealLm
```

Если аквилон нужен и снизу по выбранному узлу, добавить нижнюю линию.

### Утеплитель-окантовка

```
openingPerimeterM = 2 * (openingWidthMm + openingHeightMm) / 1000
borderAreaM2 = openingPerimeterM * 0.15
```

## 10. Что требует уточнения перед финальной инженерной версией

Пока не подтверждены окончательно:

- точный тип профиля рамки проема для каждой подсистемы и каждого узла;
- необходимость отдельных кронштейнов по каждому профилю рамки;
- шаг крепления фасонных элементов;
- перехлест фасонных элементов при длине более `3 м`;
- как считать доборные кассеты вокруг проема без координат проема;
- отличаются ли правила для дверей, ворот и окон в выбранных узлах ИНСИ.
