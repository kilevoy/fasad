```json
  "orientation": "vertical | horizontal",
  "baseCassetteFamily": "kf1",

  "code": "2178",
  "price": 1650,
  "unit": "m2",

  "status": "active"

```

## 3. Правило матчинга

Искать по полям:
- `productType`
- `family`
- `orientation`
- `thickness`
- `coating`

Примеры:
- `ukf1 + vertical + 0.7 + polyester` -> `2178`
- `ukf4_30 + vertical + 1.2 + colorflow_1s` -> `71723`
- `ugkf1 + horizontal + 0.7 + polyester` -> `2180`

## 5. Что еще надо добрать

Пока не довынесены в seed:
- `УГКФ3`
- `УГКФ4/17`
- `УГКФ4/30`
- полные наборы покрытий `colorflow_2s` для всех горизонтальных вариантов
- размерные параметры `Lлев`, `Lправ`, `Hверх`, `Hниз` как отдельные инженерные поля

## 6. Что важно для расчета

Прайсовая позиция угловой кассеты не заменяет инженерные ограничения:
- диапазоны сторон;
- увеличенные размеры по согласованию;
- дополнительное крепление при `L > 500 мм`;
- различие между обычной и горизонтальной угловой кассетой.

Эти правила должны оставаться в расчетном слое независимо от цены.
