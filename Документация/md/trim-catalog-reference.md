# Нормализованный Справочник Фасонных Элементов

Этот файл связывает комплектующие и доборные элементы с нормализованными прайсовыми позициями.

## 1. Базовые семейства

| Семейство | Что это |
| --- | --- |
| `outer_angle_complex` | Угол наружный `УСНс` |
| `inner_angle_complex` | Угол внутренний `УСВс` |
| `aquilon` | Аквилон `Акс` |
| `cover_strip` | Нащельник `Нс` |
| `casing` | Наличник `НЛс` |
| `top_flashing` | Отлив верхний `Овс` |
| `drip` | Водоотлив `Вс...` |
| `fixing_strip` | Планка крепежная `КПс` |
| `starter_strip` | Планка начальная `НПс` |
| `starter_strip_kf2` | Планка начальная `ПНКФ2` |
| `starter_strip_kf3` | Планка начальная `ПНКФ3` |
| `slope_element` | Откосный элемент `ОЭк...` |
| `decorative_slope` | Откос декоративный `ОДс` |

## 2. Ключевые поля

```json
{
  "productType": "trim",
  "family": "aquilon",
  "mark": "Акс",
  "standardLengthMm": 3000,
  "thickness": 0.5,
  "coating": "polyester | colorflow_1s | colorflow_2s | galvanized",
  "unit": "pcs",
  "code": "2168",
  "price": 502,
  "weight": 1.558,
  "status": "active"
}
```

## 3. Правило матчинга

Искать по полям:
- `productType`
- `family`
- `mark`
- `sizeVariant`, если он есть
- `thickness`
- `coating`
- `standardLengthMm`

Примеры:
- `aquilon + Акс + 0.5 + polyester + 3000` -> `2168`
- `drip + Вс180 + 180 + 0.5 + polyester + 3000` -> `2170`
- `starter_strip_kf2 + ПНКФ2 + 0.7 + polyester + 3000` -> `2649`

## 4. Что уже включено

Уже вынесены базовые позиции по:
- углам `УСНс`, `УСВс`
- аквилону
- нащельнику
- наличнику
- верхнему отливу
- водоотливам `180` и `250`
- крепежной и стартовой планке
- начальным планкам `КФ2` и `КФ3`
- откосным элементам
- декоративному откосу

## 5. Что еще надо добрать

Пока не включены:
- `УНс 50/85`
- `УВс 50/85`
- `ОПс 180/250`
- `УО180x70`
- оцинкованные и окрашенные варианты для всех семейств
- полный набор `декор` и `colorflow_2s` по всем позициям

## 6. Важное ограничение

Прайсовая строка фасонного элемента не заменяет монтажное правило:
- зона применения;
- способ стыка;
- перехлест;
- шаг крепления.

Эти параметры должны оставаться в расчетном слое и в справочнике узлов.
