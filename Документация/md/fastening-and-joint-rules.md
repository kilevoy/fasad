# Крепеж и узлы примыканий

## Статус

- Ниже собраны только правила, которые подтверждены текущими документами и/или официальной страницей ИНСИ.
- Если точный шаг крепления в документах не найден, это отмечено отдельно, без подстановки вымышленного значения.

## 1. Фасонные элементы: тип крепежа

Подтверждено по [11-103-30.md](C:\КАЛЬКУЛЯТОР КАССЕТ\Документация\md\11-103-30.md) и [albom-tehresheniy-po-fs-psch.md](C:\КАЛЬКУЛЯТОР КАССЕТ\Документация\md\albom-tehresheniy-po-fs-psch.md):

- видимое крепление фасонных комплектующих изделий:
  - оцинкованные саморезы `4,8x20`
  - окрашенная головка
  - прокладка из `ЭПДМ`
  - крепление непосредственно к облицовочному материалу
- часть фасонных изделий:
  - откосные планки
  - наличники
  - сходные элементы
  крепится к несущим профилям оцинкованными саморезами `4,2x16`
  до монтажа облицовочного материала

### Что не подтверждено / уточнено по инструкции 2017

- в альбомах `11-103-30` и `Альбом техрешений по ФС` точный шаг крепления фасонных элементов по длине не найден
- в `Instr_M_saiding_i_panel_2017.pdf` найден отдельный шаг для части завершающих элементов:
  - нащельник в местах стыков крепится к направляющим с шагом `600 мм`
  - завершающие элементы верхнего края облицовки, включая внутренний/наружный уголок и верхний отлив, крепятся видимыми самонарезающими винтами или заклепками с шагом `400...500 мм`
- для узла `Аквилон + откосная планка` отдельный числовой шаг крепления в найденном фрагменте инструкции не указан
- для начальной планки в инструкции показано крепление к нижнему краю облицовки/профилям; отдельный числовой шаг не указан, фактически крепление привязано к расположению несущих профилей
- точная величина перехлеста фасонных элементов при сопряжении в текущих документах не найдена

## 1.2. Утвержденные рабочие шаги для калькулятора

После проверки документации ИНСИ и внешних монтажных инструкций в калькуляторе приняты рабочие правила:

| Узел / элемент | Рабочий шаг | Как считать крепеж |
| --- | --- | --- |
| Начальная планка | по несущим профилям | `1` саморез на пересечение начальной линии с вертикальным несущим профилем |
| Аквилон, боковые линии откоса | `600 мм` | `ceil(длина / 600) + 1` на каждую линию |
| Откосный элемент, боковые линии | `600 мм` | `ceil(длина / 600) + 1` на каждую линию |
| Аквилон / откосный элемент, верхняя линия | `400 мм` | `ceil(длина / 400) + 1` |
| Отлив верхний | `400 мм` | `ceil(длина / 400) + 1` |
| Водоотлив нижнего откоса | `400 мм` | `ceil(длина / 400) + 1` |
| Нащельники / боковые доборные элементы | `600 мм` | `ceil(длина / 600) + 1` |

Примечание:
- шаг `400 мм` принят для верхних и ответственных водоотводящих элементов;
- шаг `600 мм` принят для боковых линий, нащельников и аквилонов;
- если в проектной документации объекта указан другой шаг, проектный шаг имеет приоритет.

## 2. КФ-4: логика количества саморезов

Подтверждено по официальной странице ИНСИ:
- [Кассета Фасадная – КФ4, открытого крепления](https://www.insi.ru/production/fasadnye-sistemy/fasadnye-kassety/kasseta-fasadnaya-kf4/)

Уточнение по конструктивной логике крепления:
- `КФ-4` крепится по отгибкам
- `2 самореза` сверху
- `2 самореза` снизу
- итого базово `4 самореза на кассету`

Правило:
- кассета `КФ-4` имеет открытую систему крепления
- базово крепится на `4 точки крепления`
- при больших размерах кассет количество точек крепления увеличивается до `6`
- открытая система крепления позволяет монтаж как в вертикальной, так и в горизонтальной плоскости

Для калькулятора:
- базовое правило:
  - `КФ4 = 4 самореза на кассету`
  - схема: `2 сверху + 2 снизу`
- при переходе в крупный размер:
  - `КФ4 = 6 саморезов на кассету`
- рабочий порог для калькулятора принят по проектному правилу:
  - `L <= 800 мм`: `4` точки крепления
  - `L > 800 мм`: `6` точек крепления
- при появлении внутренней монтажной карты ИНСИ это правило нужно сверить и при необходимости заменить:
  - `baseFastenerPoints = 4`
  - `largeCassetteFastenerPoints = 6`
  - `largeCassetteThreshold = 800 мм`

Дополнительно подтверждено по [subsystem-types-rules.md](C:\КАЛЬКУЛЯТОР КАССЕТ\Документация\md\subsystem-types-rules.md):
- `КФ1` и `КФ4` крепятся к вертикальным направляющим
- крепеж:
  - оцинкованные саморезы
  - окрашенная шестигранная головка
  - прокладка `ЭПДМ`

## 6. Рабочие правила для калькулятора до уточнения

- фасонные элементы считать по длине и переводить в штуки по стандартной длине
- шаг крепления фасонных элементов держать отдельным справочником, который пока требует подтверждения
- `КФ-4` считать:
  - `4 точки крепления` по умолчанию
  - `6 точек крепления` для крупных размеров
  - рабочий порог крупного размера: `L > 800 мм`
- узлы примыканий считать не по одному элементу, а по выбранной схеме:
  - `Аквилон + ОПс`
  - `Аквилон + ОДс`
  - `Аквилон + УО`

## 8. Поля модели для кода

Для каждой позиции фасонного элемента в калькуляторе нужно хранить:

- `family`
- `mark`
- `nodeType`
- `fastenerType`
- `fastenerMountBase`
- `requiredLengthLm`
- `nominalLengthLm`
- `pieces`
- `fastenerStepMm`
- `fastenerStepStatus`

Где:
- `fastenerStepStatus = confirmed | needs_confirmation | user_defined`

## 9. Практическое правило на текущий этап

Пока не найден документ с фиксированным шагом крепления фасонных элементов, в MVP правильно сделать так:

- считать длину элемента
- переводить ее в `шт` по стандартной длине
- тип крепежа подставлять автоматически по узлу
- шаг крепления хранить как:
  - либо значение из ручного справочника компании
  - либо параметр, который можно будет уточнить позже без переделки логики
