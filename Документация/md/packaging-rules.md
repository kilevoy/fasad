# Правила расчета упаковки фасадных кассет

Документ фиксирует рабочую логику упаковки кассет в деревянные ящики. Это правило нужно для будущего расчета количества ящиков, габаритов упаковки и загрузки в еврофуру.

## 1. Основной принцип

Кассеты укладываются в ящик не плоской стопкой, а ставятся на ребро.

Для кассеты:

- `L` - габаритная длина кассеты по чертежу СТО;
- `H` - габаритная высота кассеты по чертежу СТО;
- `D` - глубина профиля кассеты по сечению.

В ящике:

- `L` кассеты идет поперек ящика, по ширине ящика;
- `H` кассеты идет по высоте ящика;
- `D` кассеты набирается вдоль длины ящика;
- каждая следующая кассета ставится рядом с предыдущей вдоль длинной стороны ящика.

## 2. Фиксированное правило по ящику

Длина габаритная ящика всегда:

```text
2200 мм
```

Причина: ящик ставится поперек кузова еврофуры.

Ширина ящика может варьироваться в разумных пределах под длину кассеты `L`, боковые доски, пленки, прокладки и технологические зазоры.

Высота ящика обычно двух типов:

- низкая упаковка;
- высокая упаковка.

Высоты должны подбираться так, чтобы в еврофуре можно было поставить низкую пачку на высокую и общий габарит двух пачек проходил по высоте кузова.

## 3. Расчет габарита пачки

Если кассеты стоят на ребре, габарит пачки считается так:

```text
длина пачки = количество кассет * D
ширина пачки = L
высота пачки = H
```

Для проверки в ящике:

```text
запас по длине = 2200 - длина пачки
запас по ширине = ширина ящика - L
запас по высоте = высота ящика - H
```

Эти запасы нужны под доски ящика, пленки, прокладки и технологические зазоры.

## 4. КФ-1

По СТО для КФ-1:

```text
D = 30 мм
```

С учетом допуска по чертежу:

```text
D = 30 +/- 1,5 мм
```

Для расчета упаковки пока принимаем рабочее значение:

```text
D = 30 мм
```

## 5. Проверенный пример КФ-1

Дано:

```text
тип кассеты: КФ-1
L = 1520 мм
H = 572 мм
D = 30 мм
количество = 72 шт
ящик = 2200 x 1670 x 740 мм
```

Кассеты стоят на ребре:

```text
длина пачки = 72 * 30 = 2160 мм
ширина пачки = 1520 мм
высота пачки = 572 мм
```

Габарит пачки:

```text
2160 x 1520 x 572 мм
```

Запасы в ящике:

```text
по длине: 2200 - 2160 = 40 мм
по ширине: 1670 - 1520 = 150 мм
по высоте: 740 - 572 = 168 мм
```

Вывод: такая упаковка проходит, потому что запаса хватает на деревянный ящик, пленки, прокладки и технологические зазоры.

## 6. Низкая и высокая упаковка КФ-1

Для КФ-1 зафиксированы два рабочих типа упаковки:

| Тип упаковки | Количество кассет | Принцип |
| --- | ---: | --- |
| Низкая | 72 шт | один ряд кассет на ребре |
| Высокая | 144 шт | два ряда: второй ряд ставится на первый |

Для высокой упаковки:

```text
144 шт = 72 шт в нижнем ряду + 72 шт в верхнем ряду
```

По длине ящика расчет остается таким же, как для одного ряда:

```text
72 * 30 = 2160 мм
```

По высоте высокая упаковка занимает два ряда кассет:

```text
высота пачки = 2 * H
```

Для примера `H = 572 мм`:

```text
высота пачки = 2 * 572 = 1144 мм
```

К этой высоте нужно добавить запас на доски, прокладки, пленки и разделение между рядами.

## 7. Формулы для будущего калькулятора

Для КФ-1:

```text
кассет в одном ряду = floor((2200 - запас_по_длине) / 30)
```

Практически подтвержденное значение:

```text
кассет в одном ряду = 72 шт
```

Тогда:

```text
низкая упаковка = 72 шт
высокая упаковка = 144 шт
```

Ориентировочный габарит ящика:

```text
длина ящика = 2200 мм
ширина ящика = L + запас_на_ширину
высота низкой упаковки = H + запас_на_высоту
высота высокой упаковки = 2 * H + запас_на_высоту_и_разделение
```

Точные запасы нужно уточнить по чертежу упаковки.

## 8. Что уточнить по чертежу упаковки

После получения чертежа упаковки нужно зафиксировать:

- толщину досок ящика;
- фактические внутренние и наружные габариты ящика;
- минимальный запас по ширине;
- минимальный запас по высоте;
- прокладки между кассетами;
- прокладку или разделитель между первым и вторым рядом для высокой упаковки;
- допустимую высоту высокой и низкой пачки для совместной загрузки в еврофуру.

## 9. Геометрия типов кассет по СТО

Для упаковки важно разделять:

- `L` - наружный габарит кассеты по длине;
- `H` - наружный габарит кассеты по высоте;
- `D` - глубина профиля, то есть размер, который набирается вдоль ящика, если кассеты стоят на ребре;
- `Lp`, `Hp`, `H1`, `H2` - монтажные или внутренние размеры по чертежу, не являются наружным габаритом упаковки.

Для габарита пачки в ящике используются именно:

```text
ширина пачки = L
высота пачки = H
длина пачки = количество кассет в ряду * D
```

### 9.1 Сводная таблица

| Тип кассеты | Глубина `D` для упаковки | Связь размеров с `L` и `H` |
| --- | ---: | --- |
| КФ-1 | `30 +/- 1,5 мм` | `L1 = L - 47`; `Lp = L - 15`; `H1 = H - 47`; `H2 = H - 55`; `Hp = H - 25` |
| КФ-2 | `30 +/- 1,5 мм` | `Lp = L - 30`; `Hp = H - 20` |
| КФ-3 | `17 +/- 1,5 мм` | `Lp = L - 30`; `Hp = H - 18`; `H1 = H - 25` |
| КФ-3-05 | `17 +/- 1,5 мм` | `Lp = L - 30`; `Hp = H - 18`; `H1 = H - 25` |
| КФ-4/30 | `30 мм` | `Lp = L - 25`; `Hp = H - 19`; `H1 = H - 47` |
| КФ-4/17 | `17 мм` | `Lp = L - 25`; `Hp = H - 14`; `H1 = H - 34` |

Примечание: для КФ-4/30 и КФ-4/17 в СТО на сечении указаны соответственно `30` и `17`. Допуск на наружные `L/H` указан отдельно как `+/- 2,0 мм`.

## 10. Предварительный вывод для упаковки

Для упаковки на ребре все типы можно считать по одной схеме:

```text
длина ящика = 2200 мм
ширина пачки = L
высота пачки = H
длина пачки = количество * D
```

Для типов с глубиной `30 мм` рабочий ряд по аналогии с КФ-1:

```text
72 шт * 30 мм = 2160 мм
остаток по длине ящика = 40 мм
```

К типам с глубиной `30 мм` относятся:

- КФ-1;
- КФ-2;
- КФ-4/30.

Для типов с глубиной `17 мм` временно принимаем расчет по полному габариту длины ящика:

```text
2200 / 17 = 129,41 шт
129 шт * 17 мм = 2193 мм
остаток по длине ящика = 7 мм
```

К типам с глубиной `17 мм` относятся:

- КФ-3;
- КФ-3-05;
- КФ-4/17.

Временное рабочее значение:

```text
кассет в одном ряду = 129 шт
```

Это значение нужно позже уточнить по чертежу упаковки, прокладкам и допустимому минимальному запасу по длине.
