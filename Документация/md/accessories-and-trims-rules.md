# Правила по комплектующим и доборным элементам

Этот файл хранит рабочие правила для расчета комплектующих и доборных
элементов в калькуляторе фасадных кассет и подсистемы.

Сюда заносится подтвержденная информация по уголкам, планкам, отливам,
нащельникам, откосным и декоративным элементам, а также по способу их учета
в спецификации.

## Общие принципы

- Комплектующие и доборные элементы считаются отдельно от рядовых кассет.
- Комплектующие нельзя смешивать с угловыми кассетами.
- Наружные и внутренние элементы должны храниться отдельными позициями.
- Элементы с разной геометрией и разной шириной заготовки должны быть разными позициями.
- Базовая единица расчета для большинства комплектующих — `погонный метр`.
- После расчета по длине результат должен уметь переводиться в штуки по стандартной длине поставки.

## Что подтверждено источниками

Из инструкции по монтажу:

- до начала монтажа устанавливаются угловые элементы
- подгонка фасада может выполняться за счет комплектующих элементов:
- откосные углы
- наружные уголки
- внутренние уголки

Из ТУ по панелям, сайдингу и комплектующим:

- подтверждены отдельные комплектующие профили и уголки
- подтверждены их обозначения
- подтверждены толщина, стандартная длина, масса 1 п.м. и ширина заготовки

## Как считать в калькуляторе

### По длине

- уголки, планки, отливы, нащельники и откосные элементы считать по погонным метрам
- после расчета по длине переводить в штуки по стандартной длине `3 м`
- при необходимости учитывать коэффициент запаса и подрезки

### По типу

- наружные и внутренние элементы не объединять
- элементы с размерами `50` и `85` не объединять
- `УСНс/УСВс` не смешивать с `УНс/УВс`
- `ОПс`, `ОЭк`, `ОДс`, `ОВс`, `Нс`, `НЛс`, `Акс`, `НПс`, `КПс` считать как отдельные классы комплектующих

## Что хранить в модели данных

Для каждой позиции:

- код/обозначение
- полное наименование
- тип элемента
- наружный или внутренний вариант, если применимо
- типоразмер
- толщина
- стандартная длина
- масса 1 п.м.
- ширина заготовки
- единица расчета

## Связь с другими разделами расчета

Комплектующие и доборные элементы должны быть связаны с:

- углами фасада
- оконными и дверными проемами
- откосами
- примыканиями
- цоколем
- верхними завершениями

## Что пока не нужно придумывать

Пока не подтверждены и не должны считаться окончательно установленными:

- автоматическое правило выбора конкретного комплектующего без участия пользователя
- нормы крепежа по каждому типу комплектующего
- правила запасов для каждого типа элемента, если это не задано отдельно

## Рабочий вывод для MVP

Для первой версии калькулятора нужно заложить:

- отдельный справочник комплектующих и доборных элементов
- расчет по погонным метрам
- перевод в штуки по длине `3 м`
- раздельный учет наружных и внутренних элементов
- раздельный учет разных типоразмеров и профилей
