# Нормализованный Справочник Крепежа

Этот файл связывает крепежные позиции прайса с инженерными правилами расчета.

Связанные файлы:
- [fastening-and-joint-rules.md](C:\КАЛЬКУЛЯТОР КАССЕТ\Документация\md\fastening-and-joint-rules.md)
- [fastener-catalog.seed.json](C:\КАЛЬКУЛЯТОР КАССЕТ\Документация\data\fastener-catalog.seed.json)

## 1. Базовые семейства

| Семейство | Что это |
| --- | --- |
| `rivet` | Заклепки |
| `self_drilling_screw` | Самосверлящие саморезы |
| `facade_dowel` | Фасадный дюбель |
| `nail_dowel` | Дюбель-гвоздь |
| `bracket_washer` | Шайба усиления кронштейна |

## 2. Ключевые поля

```json
{
  "productType": "fastener",
  "family": "self_drilling_screw",
  "mark": "4_8x20_ral",
  "diameter": 4.8,
  "length": 20,
  "finish": "ral",
  "unit": "pcs",
  "code": "2337",
  "price": 3.24,
  "status": "active"
}
```

## 3. Правило матчинга

Искать по полям:
- `productType`
- `family`
- `diameter`
- `length`
- `finish`
- `headType`, если он используется

Примеры:
- `self_drilling_screw + 4.2 + 16 + galvanized` -> `1800`
- `self_drilling_screw + 4.8 + 20 + ral` -> `2337`
- `rivet + 4.8 + 8 + plain` -> `10367`

## 4. Что уже включено

Уже вынесены:
- базовые заклепки подсистемы;
- окрашенные и оцинкованные саморезы `4,2x16`, `4,8x20`, `4,8x35`, `4,8x70`;
- фасадный дюбель `RDF 10x100 HEX`;
- дюбель-гвозди `8x80`, `8x100`;
- `Ш50`.

## 5. Что еще надо добрать

Пока не вынесены в общий fastener seed:
- специализированные саморезы `HR/HW/HGX/HWX`;
- часть крепежа с прокладками `RUSPERT`;
- тарельчатые дюбели утеплителя как прайсовые позиции;
- анкеры / химические анкеры как полный блок;
- шаг крепления для фасонных элементов.

## 6. Важное ограничение

Сам seed по крепежу не заменяет норму расхода.

Норма расхода продолжает жить в:
- правилах по кассетам;
- правилах по утеплителю;
- правилах по узлам;
- правилах по подсистеме.
