# Подсистема На Углах

## 1. Общий принцип

Подсистема в зоне угла не равна обычной рядовой зоне фасада.

Для калькулятора нужно отдельно учитывать:

- `наружные углы`;
- `внутренние углы`;
- тип подсистемы:
  - `П-образная одноуровневая`;
  - `П-образная двухуровневая`;
  - `Г-образная`;
- тип облицовочного решения:
  - угловая кассета;
  - уголок;
  - внутренний уголок;
  - кассетный внутренний угол, если он подтвержден узлом.

## 2. П-образная система на углах

### 2.1. Наружный угол

Подтверждено по альбому:

- есть отдельный узел наружного угла при `вертикальной выравнивающей системе`;
- есть отдельный узел наружного угла при `горизонтальной выравнивающей системе`;
- ближайший профиль у наружного угла ставится не ближе `130 мм` от линии угла.

## 3. Г-образная система на углах

### 3.1. Наружный угол

Подтверждено:

- наружный угол в `Г-образной` системе является отдельным узлом подсистемы;
- для наружного угла применяется отдельный `Z`-профиль;
- в прочностной выжимке отдельно подтвержден `Z 40x21x20` для оформления внешних углов.

Источники:

Рабочее правило калькулятора:

- на каждый наружный угол учитывать отдельный вертикальный `Z`-профиль по высоте угла.

### 3.2. Внутренний угол

Для `Г-образной` системы внутренний угол в текущей модели не детализирован отдельным специальным профилем с таким же уровнем подтверждения, как наружный угол.

Рабочее правило:

- пока не вводить отдельный специальный профиль внутреннего угла `Г`-системы без доп. подтверждения.

## 4. Связь подсистемы и облицовки на углах

Подсистема на углу и облицовка угла — это не одно и то же.

Нужно различать:

- `угловую кассету` как облицовочный элемент;
- `уголок` как фасонный элемент;
- `дополнительные профили подсистемы` в зоне угла.

То есть даже если наружный угол оформлен `угловой кассетой`, в подсистеме у угла все равно может требоваться отдельная угловая организация профилей.

## 5. Учет выноса подсистемы в угловых кассетах

В калькуляторе введена первая инженерная модель влияния выноса подсистемы на размеры угловых кассет.

Расчетный вынос угла принимается как:

- для `П-образной одноуровневой` системы: `толщина утеплителя + воздушный зазор + 27 мм`;
- для `П-образной двухуровневой` системы: `толщина утеплителя + воздушный зазор + 20 мм`;
- для `Г-образной` системы: `толщина утеплителя + воздушный зазор + 20 мм`.

Эта величина прибавляется к расчетной полке угловой кассеты. Поэтому при увеличении утеплителя, воздушного зазора или смене схемы подсистемы:

- размеры полок угловых кассет;
- количество рядовых кассет между углами;
- подбор экономичного варианта по стандартной длине или высоте.

## 6. Ограничение текущей модели

Текущая модель уже учитывает общий вынос облицовки, но еще не строит полный 3D-узел угла по осям всех профилей.

Для следующей версии нужно уточнить:

1. положение строительной оси угла;
2. фактическое положение осей профилей подсистемы у угла;
3. расстояние от линии угла до линии опирания кассеты;
4. отдельные правила для разных узлов угла и типов кассет.

## 7. Текущий статус

Сейчас в калькуляторе уже есть:

- отдельные угловые элементы подсистемы для `П`-системы;
- отдельный `Z`-профиль наружного угла для `Г`-системы;
- отдельный учет угловых кассет как облицовочных элементов.
- учет расчетного выноса подсистемы в размерах `L1/L2` угловых кассет;
- пересчет количества рядовых кассет между углами с учетом увеличенных угловых полок.

## 8. Практический вывод

На текущем этапе калькулятор уже умеет:

- учитывать углы в подсистеме;
- учитывать углы в облицовке;
- не смешивать эти сущности.
- увеличивать угловые кассеты на расчетный вынос подсистемы;
- уменьшать поле рядовых кассет между углами на этот же расчетный вынос.
