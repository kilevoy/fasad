# Калькулятор фасадных кассет

Стартовый бесплатный MVP под `GitHub Pages`.

## Стек

- `React`
- `TypeScript`
- `Vite`
- `GitHub Actions`
- `GitHub Pages`

## Задача этого каркаса

- держать расчетное ядро отдельно от интерфейса;
- поддержать кассеты `КФ-1`...`КФ-4`;
- подготовить модули под подсистему, утеплитель, пленку, саморезы и заклепки;
- позже подключить прайс из Excel без переписывания логики расчета.

## Команды

```bash
npm install
npm run dev
npm run build
```

## Публикация на GitHub Pages

В репозитории уже добавлен workflow:

- `.github/workflows/deploy.yml`

Для публикации нужно:

1. Залить проект в GitHub-репозиторий.
2. Работать из ветки `main`.
3. В настройках репозитория включить `GitHub Pages` с источником `GitHub Actions`.
