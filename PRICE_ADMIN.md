# Обновление общего прайса

Общий прайс хранится в `public/price.json` и автоматически загружается на сайте GitHub Pages.

## Как обновить

### Вариант 1. Через файл `update-price.bat`

Самый простой способ:

1. Открыть папку `web`.
2. Запустить `update-price.bat` двойным кликом.
3. Перетащить новый Excel-прайс в открывшееся окно или нажать Enter для файла по умолчанию.
4. Дождаться сообщения `Готово. Прайс отправлен на GitHub`.

Скрипт сам выполнит импорт, проверку сборки, проверку линтера, коммит и `git push`.

### Вариант 2. Вручную через PowerShell

```powershell
cd "C:\Users\Deako\Мой диск\КАЛЬКУЛЯТОР КАССЕТ\web"
npm run price:import -- "C:\Users\Deako\Мой диск\КАЛЬКУЛЯТОР КАССЕТ\прайс март2026.xlsx"
npm run build
npm run lint
git add package.json scripts/import-price.mjs public/price.json src/App.tsx src/features/calculator/model/price-upload.ts PRICE_ADMIN.md
git commit -m "Update shared price list"
git push origin main
```

После `git push` GitHub Pages пересоберет сайт. Обычно обновление занимает 1-3 минуты.

## Защита от ошибки

Команда проверяет контрольные позиции и не сохранит прайс, если вместо колонки `Цена` будет взята колонка `Цена за 1т в руб.`.

Контрольные коды:

- `2182` должен быть `1477`
- `68975` должен быть `2336`
- `118224` должен быть `12`
- `118181` должен быть `33`

## Локальный прайс менеджера

Кнопка `Загрузить прайс` на сайте остается для временной проверки Excel в браузере. Такой прайс имеет приоритет только у конкретного пользователя. Чтобы вернуться к общему прайсу, нужно нажать `Сбросить локальный`.
